# Nadie Publico

Portada publica de `Nadie te dijo que iba a ser asi`.

## Archivos

- `index.html`: estructura de la portada.
- `styles.css`: look cinematografico, overlay oscuro, grano y luces rojas/azules por CSS.
- `script.js`: fallback basico para el video.
- `assets/video/`: video de fondo.
- `assets/images/`: poster/fallback.
- `assets/logos/`: logo de SKA Films cuando este disponible.

## Assets necesarios

Subir manualmente estos dos archivos con estos nombres exactos:

- `assets/video/ntdqiasa_fire_smoke_ash_web.mp4`
- `assets/images/ntdqiasa_fire_smoke_ash_poster.jpg`

## GitHub Pages

Cuando los assets esten subidos:

1. Ir a `Settings`.
2. Entrar en `Pages`.
3. Elegir `Deploy from a branch`.
4. Branch: `main`.
5. Folder: `/root`.
6. Guardar.
